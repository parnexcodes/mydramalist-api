## API Reference

```http
  GET /api/${type}/${filter}?page=${pagenum}
```

| Parameter | Type     | Description                       |
| :-------- | :------- | :-------------------------------- |
| `type`      | `string` | **Required**, can be `shows` or `movies` |
| `filter`      | `string` | **Required**, can be `top`, `popular`, `newest`, `upcoming`, `variety` |
| `pagenum`      | `string` | **Optional**, eg: `2`|
